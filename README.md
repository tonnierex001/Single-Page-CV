# Single-Page-CV
The goal of this project is to learn how to create a structured, single-page CV using only HTML. I will focus on laying out my education, skills, and career history in a clean, semantic manner. This is not my comprehensive CV; dummy placeholders were used in place of real instructions.

Instructions to Run the Code:
Preview the code on any browser of your choice.
